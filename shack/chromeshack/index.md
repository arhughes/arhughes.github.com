## Features

* LOL'ing (with customizable tags)
* Thread expiration timer
* Comments tags
* Post preview
* Highlight users (mod/employee/dev/custom)
* Highlight new comments
* Category banners
* Image loader
* Video loader
* Collapse threads
* Gegtik comic strip
* Sparkly comic strip
* Auto-updating
